# buildToolsFor_cpp_c <br/>
basic build tools premade for building and compiling cpp and c <br/>
commands: <br/>
    Makefile: make <br/>
    Cmake: cmake #(from folder were you want build to folder with the CMakeLists.txt)<br/>
           cmake --build . #(in folder for build)<br/>
    tasks:just run them with ctrl-shift-b or from tasks(in search bar) in vscode or vs<br/>
